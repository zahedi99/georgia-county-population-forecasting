
This project builds a county-level population forecasting pipeline for the U.S. state of Georgia using historical Census estimates and machine learning models.

The system:
- processes multi-year demographic time-series data
- engineers temporal growth features
- trains forecasting models per county
- generates 10-year projections
- produces interpretable outputs and comparison summaries

Outputs include:
- per-county forecasts
- statewide projections
- model comparison tables
- visualization of population growth trends

Tech stack:
Python • Pandas • Scikit-learn • XGBoost • Time-series feature engineering
